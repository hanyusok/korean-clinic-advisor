# 사용자 스토리 (User Stories)

## 사용자 페르소나

### 페르소나 1: 미용 관광객 (Sarah, 28세, 싱가포르)
- **목적**: 한국 여행 중 피부 시술을 받고 싶음
- **요구사항**: 
  - 영어로 정보를 확인하고 싶음
  - 가격을 미리 비교하고 싶음
  - 신뢰할 수 있는 클리닉을 찾고 싶음
- **기술 수준**: 스마트폰 사용에 익숙함

### 페르소나 2: 반복 방문자 (Chen, 35세, 중국)
- **목적**: 이전에 방문했던 클리닉을 다시 찾거나 새로운 클리닉 탐색
- **요구사항**:
  - 이전 방문 기록 확인
  - 할인 정보 확인
  - 위치 기반 검색
- **기술 수준**: 모바일 앱 사용에 매우 익숙함

### 페르소나 3: 클리닉 관리자 (김대표, 45세)
- **목적**: 클리닉 정보를 관리하고 고객을 유치
- **요구사항**:
  - 클리닉 정보 업데이트
  - 가격 정보 관리
  - 리뷰 모니터링

## 사용자 스토리

### 검색 및 탐색

#### US-001: 지역별 클리닉 검색
**As a** 미용 관광객  
**I want to** 강남 지역의 클리닉을 검색할 수 있어야 함  
**So that** 내가 머무는 지역 근처의 클리닉을 쉽게 찾을 수 있음

**Acceptance Criteria:**
- 지역 필터를 선택할 수 있음 (강남, 청담, 압구정 등)
- 선택한 지역의 클리닉 목록이 표시됨
- 각 클리닉 카드에 기본 정보가 표시됨 (이름, 평점, 위치)

#### US-002: 시술 종류별 검색
**As a** 미용 관광객  
**I want to** 원하는 시술 종류로 클리닉을 필터링할 수 있어야 함  
**So that** 내가 원하는 시술을 제공하는 클리닉만 볼 수 있음

**Acceptance Criteria:**
- 시술 카테고리 선택 가능 (레이저, 보톡스, 스킨케어 등)
- 선택한 시술을 제공하는 클리닉만 표시됨
- 각 클리닉의 해당 시술 가격이 표시됨

#### US-003: 가격대별 검색
**As a** 예산이 정해진 관광객  
**I want to** 예산 범위에 맞는 클리닉을 검색할 수 있어야 함  
**So that** 내 예산 내에서 최적의 클리닉을 찾을 수 있음

**Acceptance Criteria:**
- 최소/최대 가격 범위를 설정할 수 있음
- 설정한 가격 범위 내의 클리닉만 표시됨
- 가격 정보가 명확하게 표시됨

#### US-004: 통합 검색
**As a** 사용자  
**I want to** 클리닉명, 지역, 시술 종류를 한 번에 검색할 수 있어야 함  
**So that** 빠르게 원하는 클리닉을 찾을 수 있음

**Acceptance Criteria:**
- 검색어 입력 필드가 있음
- 검색어에 맞는 클리닉이 실시간으로 표시됨
- 검색 결과에 하이라이트 표시

### 클리닉 정보 조회

#### US-005: 클리닉 상세 정보 확인
**As a** 관광객  
**I want to** 클리닉의 상세 정보를 확인할 수 있어야 함  
**So that** 클리닉에 대한 충분한 정보를 바탕으로 결정할 수 있음

**Acceptance Criteria:**
- 클리닉명, 주소, 전화번호가 표시됨
- 운영시간 정보가 표시됨
- 클리닉 사진을 갤러리로 볼 수 있음
- 지도에서 위치를 확인할 수 있음

#### US-006: 시술 가격 정보 확인
**As a** 가격을 비교하고 싶은 관광객  
**I want to** 클리닉별 시술 가격을 확인할 수 있어야 함  
**So that** 여러 클리닉의 가격을 비교할 수 있음

**Acceptance Criteria:**
- 시술 종류별 가격이 표시됨
- 가격이 명확하게 표시됨 (KRW, USD)
- 패키지 상품 정보가 표시됨

#### US-007: 지도에서 클리닉 위치 확인
**As a** 위치를 확인하고 싶은 관광객  
**I want to** 지도에서 클리닉의 정확한 위치를 확인할 수 있어야 함  
**So that** 내 숙소나 현재 위치에서 얼마나 가까운지 알 수 있음

**Acceptance Criteria:**
- 지도 뷰에서 클리닉 위치가 표시됨
- 현재 위치와의 거리가 표시됨
- 대중교통 경로를 확인할 수 있음 (향후 확장)

### 리뷰 시스템

#### US-008: 리뷰 읽기
**As a** 클리닉을 선택하려는 관광객  
**I want to** 다른 사용자들의 리뷰를 읽을 수 있어야 함  
**So that** 클리닉에 대한 실제 경험을 바탕으로 결정할 수 있음

**Acceptance Criteria:**
- 리뷰 목록이 표시됨
- 리뷰를 평점, 최신순으로 정렬할 수 있음
- 리뷰에 사진이 포함될 수 있음
- 리뷰가 도움이 되었는지 표시할 수 있음

#### US-009: 리뷰 작성
**As a** 클리닉을 방문한 사용자  
**I want to** 내 경험에 대한 리뷰를 작성할 수 있어야 함  
**So that** 다른 사용자들에게 도움을 줄 수 있음

**Acceptance Criteria:**
- 로그인한 사용자만 리뷰 작성 가능
- 별점(1-5점)을 선택할 수 있음
- 리뷰 내용을 작성할 수 있음
- 사진을 업로드할 수 있음 (선택)
- 방문 날짜를 선택할 수 있음

#### US-010: 리뷰 수정/삭제
**As a** 리뷰를 작성한 사용자  
**I want to** 내가 작성한 리뷰를 수정하거나 삭제할 수 있어야 함  
**So that** 잘못 작성한 리뷰를 수정할 수 있음

**Acceptance Criteria:**
- 내가 작성한 리뷰만 수정/삭제 가능
- 수정 시 기존 내용이 표시됨
- 삭제 전 확인 메시지가 표시됨

### 사용자 계정

#### US-011: 회원가입 및 로그인
**As a** 새 사용자  
**I want to** 소셜 로그인으로 쉽게 가입할 수 있어야 함  
**So that** 빠르게 서비스를 이용할 수 있음

**Acceptance Criteria:**
- Google, Kakao 소셜 로그인 지원
- 이메일 회원가입 지원
- 로그인 상태가 유지됨

#### US-012: 마이페이지
**As a** 로그인한 사용자  
**I want to** 내가 작성한 리뷰와 즐겨찾기를 확인할 수 있어야 함  
**So that** 내 활동을 관리할 수 있음

**Acceptance Criteria:**
- 내가 작성한 리뷰 목록이 표시됨
- 즐겨찾기한 클리닉 목록이 표시됨
- 프로필 정보를 수정할 수 있음

#### US-013: 즐겨찾기
**As a** 관심 있는 클리닉을 찾은 사용자  
**I want to** 클리닉을 즐겨찾기에 추가할 수 있어야 함  
**So that** 나중에 쉽게 다시 찾을 수 있음

**Acceptance Criteria:**
- 클리닉 상세 페이지에서 즐겨찾기 버튼 클릭
- 즐겨찾기에 추가/제거 가능
- 마이페이지에서 즐겨찾기 목록 확인

### 다국어 지원

#### US-014: 언어 전환
**As a** 외국인 관광객  
**I want to** 내가 이해하는 언어로 정보를 볼 수 있어야 함  
**So that** 언어 장벽 없이 서비스를 이용할 수 있음

**Acceptance Criteria:**
- 언어 선택 메뉴가 있음
- 한국어, 영어, 중국어, 일본어 지원
- 선택한 언어로 모든 콘텐츠가 표시됨

### 관리자 기능

#### US-015: 클리닉 정보 관리
**As a** 클리닉 관리자  
**I want to** 클리닉 정보를 등록하고 수정할 수 있어야 함  
**So that** 정확한 정보를 제공할 수 있음

**Acceptance Criteria:**
- 관리자 권한이 필요함
- 클리닉 기본 정보 입력/수정
- 가격 정보 입력/수정
- 이미지 업로드

#### US-016: 리뷰 관리
**As a** 관리자  
**I want to** 부적절한 리뷰를 삭제하거나 숨길 수 있어야 함  
**So that** 서비스의 신뢰성을 유지할 수 있음

**Acceptance Criteria:**
- 부적절한 리뷰 신고 기능
- 관리자가 리뷰 삭제/숨김 가능
- 삭제 사유 기록
