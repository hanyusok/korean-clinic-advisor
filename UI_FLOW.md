# UI 플로우 및 와이어프레임

## 주요 사용자 플로우

### 플로우 1: 클리닉 검색 및 상세 조회

```
홈페이지
  ↓
검색 바에 "강남 레이저" 입력
  ↓
검색 결과 페이지
  - 필터 사이드바 (지역, 시술, 가격)
  - 클리닉 카드 리스트
  ↓
클리닉 카드 클릭
  ↓
클리닉 상세 페이지
  - 탭: Overview / Treatments / Reviews / Location
  ↓
Reviews 탭 클릭
  ↓
리뷰 목록 확인
  ↓
리뷰 작성 버튼 클릭 (로그인 필요)
  ↓
리뷰 작성 페이지
```

### 플로우 2: 리뷰 작성

```
클리닉 상세 페이지
  ↓
"리뷰 작성" 버튼 클릭
  ↓
로그인 확인
  - 미로그인: 로그인 페이지로 이동
  - 로그인: 리뷰 작성 폼 표시
  ↓
리뷰 작성 폼
  - 클리닉 선택 (자동 채워짐)
  - 시술 종류 선택
  - 별점 선택 (1-5)
  - 방문 날짜 선택
  - 리뷰 내용 입력
  - 사진 업로드 (선택)
  ↓
제출
  ↓
리뷰 작성 완료
  ↓
클리닉 상세 페이지로 리다이렉트
```

### 플로우 3: 즐겨찾기

```
클리닉 상세 페이지
  ↓
하트 아이콘 클릭
  ↓
즐겨찾기 추가/제거
  ↓
마이페이지
  ↓
즐겨찾기 탭
  ↓
저장한 클리닉 목록 확인
```

## 컴포넌트 설계

### 공통 컴포넌트

#### ClinicCard
```tsx
<ClinicCard
  id={clinic.id}
  name={clinic.name}
  image={clinic.mainImage}
  rating={clinic.rating}
  reviewCount={clinic.reviewCount}
  address={clinic.address}
  priceRange={clinic.priceRange}
  region={clinic.region}
/>
```

#### ReviewCard
```tsx
<ReviewCard
  id={review.id}
  userName={review.user.name}
  rating={review.rating}
  content={review.content}
  visitDate={review.visitDate}
  images={review.images}
  helpfulCount={review.helpfulCount}
  treatment={review.treatment}
/>
```

#### SearchBar
```tsx
<SearchBar
  placeholder="클리닉명, 지역, 시술 검색..."
  onSearch={(query) => handleSearch(query)}
  suggestions={suggestions}
/>
```

#### FilterSidebar
```tsx
<FilterSidebar
  regions={regions}
  treatments={treatments}
  priceRange={priceRange}
  onFilterChange={(filters) => handleFilter(filters)}
/>
```

## 반응형 디자인

### 모바일 (< 640px)
- 필터 사이드바: 하단 시트 또는 모달
- 클리닉 카드: 전체 너비, 세로 레이아웃
- 이미지 갤러리: 스와이프 가능
- 탭 메뉴: 스크롤 가능한 가로 스크롤

### 태블릿 (640px - 1024px)
- 필터 사이드바: 좌측 고정 또는 토글
- 클리닉 카드: 2열 그리드
- 이미지 갤러리: 그리드 레이아웃

### 데스크톱 (> 1024px)
- 필터 사이드바: 좌측 고정
- 클리닉 카드: 3-4열 그리드
- 이미지 갤러리: 그리드 레이아웃
- 지도: 사이드 바이 사이드
